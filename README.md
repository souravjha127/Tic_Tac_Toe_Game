Tic Tac Toe Pro 🎮
A modern take on the classic Tic Tac Toe game, built with Python and Streamlit. Challenge your friends or play against the computer with an intelligent AI opponent, all in a sleek and interactive web interface.

Features
Play vs Computer or Friends – Single-player with smart AI or two-player local mode.

Smart AI – Computer adapts to your moves for a fun and challenging experience.

Interactive UI – Built with Streamlit for smooth, easy-to-use gameplay.

Score Tracking – Keep track of wins, losses, and draws.

Customizable Visuals – Add backgrounds and enjoy vibrant colors during play.
